# Two Sum

## Problem
Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to `target`.

## Solution Approach
- **HashMap** for O(n) time complexity
- Store seen numbers and their indices

## Complexity
- Time: **O(n)**
- Space: **O(n)**

## Interview Tips for Leads
- Discuss brute force O(n²) first
- Handle duplicates and edge cases
- Follow-up: 3Sum, 4Sum
